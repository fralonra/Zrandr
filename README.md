# Zrandr
A simple PyQt front end for xrandr.  

Features include:  
* Setting screen resolution
* Adding unrecognized resolutions
* Rotating screen
* Changing screen brightness

## Dependencies
* qt5
* xrandr

## Installation
### Archlinux
```
yaourt zrandr
```
### Other Distributions
running 'install.sh' as root
```
# install.sh
```
